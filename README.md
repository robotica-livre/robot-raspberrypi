# robot-raspberrypi
Projeto de Robô baseado em Raspberry Pi. O objetivo é criar um Seguidor de linha simples com um software escrito em Python e na medida que o projeto vai evoluindo vou adicionar mais módulos.  

# Peças Utilizadas
* Raspberry Pi (Model B/B+ ou Pi 2)
* Chassi Colorado
* RaspiRobot Board
* Prototype Shield para Raspberry Pi
* Módulo seguidor de linha Keyes

#Hardware
O hardware é simples e pode ser comprado em qualquer loja. Todo o cérebro ficará concentrado no Raspberry Pi e ele terá conexão por Wifi e Bluetooth.

#Software
A linguagem usada foi o Python, uma vez que a biblioteca do RaspiRobot Board foi escrita nessa linguagem.
